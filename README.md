# 🍳 AI Recipe Generator

An AI-powered recipe generator built using React and the Hugging Face Inference API.
This application allows users to enter ingredients they already have and receive AI-generated recipe suggestions instantly.

---

# 🚀 Features

* Dynamic ingredient input system
* AI-generated recipes using Hugging Face models
* Built with React and Vite
* Responsive user interface
* Conditional rendering for better UX
* Reusable React components
* Markdown-formatted recipe responses
* Fast and lightweight frontend setup

---

# 🛠️ Tech Stack

## Frontend

* React.js
* Vite
* JavaScript (ES6+)
* CSS3

## AI Integration

* Hugging Face Inference API
* Large Language Model (LLM)

---

# 🧠 Project Functionality

Users can:

1. Add ingredients dynamically
2. Store ingredient lists using React `useState`
3. Send ingredients to a Hugging Face AI model
4. Receive recipe suggestions generated by AI
5. View formatted recipe responses directly in the UI

---

# ⚛️ React Concepts Used

## useState

The project heavily uses React `useState` for:

* Managing ingredient input
* Updating ingredient lists
* Storing generated recipes
* Dynamic UI rendering

## Conditional Rendering

Used to:

* Show recipe section only after enough ingredients are added
* Improve interaction flow

## Component-Based Architecture

The application is divided into reusable components for scalability and maintainability.

---

# 🤖 Hugging Face Model Integration

The application integrates with the Hugging Face Inference API through the `ai.js` file.

The AI model:

* Understands ingredient combinations
* Suggests practical recipes
* Generates markdown-formatted responses
* Can recommend additional ingredients when required

Example workflow:

1. User enters ingredients
2. Ingredients are passed to the Hugging Face API
3. AI processes the prompt
4. Generated recipe is displayed on screen

# ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/your-username/your-repository-name.git
```

Move into the project directory:

```bash
cd your-repository-name
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```

---

# 🔑 Environment Variables

Create a `.env` file in the root directory:

```env
VITE_HF_API_KEY=your_huggingface_api_key
```

---

# 📦 Build for Production

```bash
npm run build
```
